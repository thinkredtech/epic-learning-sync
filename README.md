# Epic Learning Sync for LearnPress
## What is it?
A simple WordPress plugin called Epic Learning Sync for LearnPress.

## What does it do?
The plugin synchronizes and populates LearnPress courses with courses data from the Epic Learning Network API.

## What else does it do?
The plugin also provides a tool to delete all courses from Learnpress.

## How can I install it?
1. Download the zip file.
2. On the WordPress Admin Dashboard, go to Plugins -> Add new plugin.
   <br/><img width="150" alt="image" src="https://github.com/user-attachments/assets/fdc68048-9d09-462a-8c31-c5caf0696f40" />
3. Click the Upload Plugin button at the top of the page
   <br/><img width="450" alt="image" src="https://github.com/user-attachments/assets/cd586941-19e8-4e0e-81ae-20938133d246" />
4. Click the `Choose File` button and select the downloaded zip file.
   <br/><img width="450" alt="image" src="https://github.com/user-attachments/assets/557fd6b0-6b3c-430b-a348-87c1361d6856" />
5. Click `Install Now`.
   <br/><img width="450" alt="image" src="https://github.com/user-attachments/assets/c4349098-62e6-47c2-b093-690d05d1bac9" />
6. Click the `Activate Plugin` button. This should take you the Installed Plugins page with the success message banner.
   <br/><img width="450" alt="image" src="https://github.com/user-attachments/assets/8d0b0371-fe59-433e-ab21-1d11e8e73ae8" />
7. The plugin should now be listed in the plugins list as Activated.
   <br/><img width="450" alt="image" src="https://github.com/user-attachments/assets/0909a275-7434-4716-a674-948e2b2f6721" />

## How can I use it?
1. On the WordPress Admin Dashboard, go to Settings -> Epic Learning Sync.
   <br/><img width="150" alt="image" src="https://github.com/user-attachments/assets/6b160572-574f-44a0-8961-dd58031fac94" />
2. This should open the Epic Learning Sync settings page.
   <br/><img width="450" alt="image" src="https://github.com/user-attachments/assets/6a198374-9b1f-41be-a14b-9cb0974fd89d" />
3. Add your API Application ID and API Key and click `Save`.
4. For fetching all courses from the Epic Learning API, simply click Sync Courses and let the Plugin do it's magic.
5. To clean up the LearnPress database and delete all courses, click the `Delete All Courses` button.
